## Overview

Aihoo is an innovative AI web platform designed to facilitate simultaneous interaction with multiple AI chatbots. It enhances comparative analysis and response optimization, providing users with a powerful tool to leverage various AI models effectively.

## Key Features

- **Multi-Chatbot Interaction**: interact with multiple AI chatbots simultaneously to compare responses and choose the best output.
- **Meta Chat AI**: implemented to refine chatbot outputs and identify top-performing models, significantly improving the overall user experience.
- **Startup Accelerator Participation**: Aihoo is currently part of Acceleratec, a leading start-up accelerator program in Bolivia.

## Technologies Used

- **Next.js**: The React framework for the web.
- **Vercel AI SDK**: For seamless integration of AI capabilities.
- **Tailwind CSS**: For rapid and responsive UI development.
- **PostgreSQL**: For robust data management.
